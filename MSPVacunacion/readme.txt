************************************
PRE-REQUISITOS
************************************
- Instalar Spring Tool Suite o cualquier IDE que soporte Spring Boot

************************************
PASOS 
************************************
1.Importar el proyecto MSPVacunacion
2.En el archivo application.properties modificar las siguientes propiedades:
	spring.datasource.password   :  Colocar el password de acceso a la base de datos.
	spring.datasource.url        :  Modificar el puerto de acceso a la base de datos, en caso de que sea diferente a 5432
	
3.Levantar el proyecto y testear los servicios rest de acuerdo a la documentación entregada.
